# seeking-os
#### ——————操作系统真象还原 - 代码复现

## 项目背景

仅个人学习，希望能提升自己，与大家共勉。  
本项目是基于郑钢所著《操作系统真象还原》一书中的内容开发的操作系统实验代码复现。本书深入浅出地讲解了操作系统的核心机制，从零开始自制操作系统全流程，并提供了详尽的代码实现与运行环境。

## 环境

- bochs版本：2.6.8  
- 编译器：gcc version 4.8.5 (Ubuntu 4.8.5-4ubuntu2)  
- 汇编器：NASM version 2.14.02  
- 链接器：GNU ld (GNU Binutils for Ubuntu) 2.34  
- make：GNU Make 4.2.1
